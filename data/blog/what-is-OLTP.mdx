## What is OLTP?

- OLTP stands for On-Line Transaction Processing

- It is a type of database system that is optimized for transactional processing, which involves inserting, updating, and reading data in a database.

- It is designed to handle a high volume of short, transactional queries, such as those needed to process credit card payments or handle customer orders. 

- OLTP systems are characterized by their high-concurrency, low latency, and high-throughput.

- It is typically implemented using a relational database management system (RDBMS) and use SQL (Structured Query Language) for data manipulation.
